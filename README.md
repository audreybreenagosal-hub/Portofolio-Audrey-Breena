# Portofolio Audrey Breena

Website portofolio sederhana untuk Audrey Breena, siswa di UPHC.

## Cara Menjalankan
1. Buka file `index.html` di browser web (Chrome, Firefox, dll.).
2. Website akan tampil dengan bagian Tentang, Proyek, dan Kontak.

## Struktur File
- `index.html`: Halaman utama.
- `style.css`: Styling untuk tampilan.
- `script.js`: JavaScript untuk interaktivitas (smooth scroll).

## Kustomisasi
- Edit `index.html` untuk mengubah konten.
- Edit `style.css` untuk mengubah desain.
- Tambahkan gambar di folder `assets/` jika diperlukan.

## Troubleshooting
- Jika tidak tampil, pastikan semua file di folder yang sama.
- Untuk hosting, upload ke GitHub Pages atau Netlify.
